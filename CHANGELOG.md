Because I don't have a macOS, I can't package dmg format.

You can only compile it manually.

Packaged files are generated to the `scrcpy/build` folder.
```shell
git clone https://github.com/Tomotoes/scrcpy-gui
cd scrcpy-gui
npm i
npm run build
```
